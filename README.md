## 🚀 Installation

### Prerequisites
- [Node.js](https://nodejs.org/) v21.7.3 or higher
- [Git](https://git-scm.com/) for cloning
- 1GB+ RAM recommended

### Step-by-step

```bash
# Clone the repository
git clone https://github.com/ayancodex503/QUEEN-NAZUMA-MINI.git

# Enter the directory
cd QUEEN-NAZUMA-MINI

# Install dependencies
npm install

# Start the bot
npm start
